# Portfólio - Apresentação profissional

Site de apresentação profissional para **Desenvolvedor Full Stack / QA**, criado com HTML, CSS e JavaScript puros — sem dependências externas ou bundlers.

## 🚀 Tecnologias

- **HTML5** — estrutura semântica
- **CSS3** — variáveis, flexbox, grid, glassmorphism, animações e design responsivo
- **JavaScript (Vanilla ES6+)** — interatividade, animações e menu mobile
- **Canvas API** — partículas animadas no fundo
- **Font Awesome 6** — ícones via CDN

## ✨ Funcionalidades

- Fundo animado com partículas conectadas por linhas (efeito "rede neural")
- Cabeçalho fixo com efeito glassmorphism e navegação suave
- Menu hambúrguer responsivo para dispositivos móveis
- Seções: Sobre, Estudos, Experiência, Competências & Idiomas, Contato
- Timeline visual para formação acadêmica e experiência profissional
- Tags interativas para habilidades e áreas de interesse
- Links de contato (e-mail, WhatsApp, GitHub, LinkedIn)

## 📁 Estrutura

```
portifolio-apresentacao/
├── index.html   # Página principal
├── style.css    # Estilos completos
├── script.js    # Interatividade e animações
└── README.md
```

## 🌐 Como usar

Basta abrir o `index.html` no navegador. Não requer instalação ou build.
